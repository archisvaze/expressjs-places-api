# Places API

This is a simple API to manage places. The API allows you to: 
- Create a place
- Get a specific place
- List places and filter them by name or city

## Installation
```
npm install the following dependencies 
"dotenv": "^16.0.1",
"express": "^4.18.1",
"mongoose": "^6.5.3",
"morgan": "^1.10.0"

devDependencies
"nodemon": "^2.0.19"

Start server using: npm run dev
```
